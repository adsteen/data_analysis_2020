Accordingly, it is important to write our code so that it is easily readable by humans. What defines easily readable? Intro to packages

2020-09-08
notes
Like all programming languages, R relies on a base language, which is extended by packages. When you installed R, you only installed the base language1. However, base R can only do so much. Most of R’s power lies is in its packages. Need to do timeseries forecasting? You’ll want the forecast package. Need to do multivariate statistics? vegan is world-leading. Currently there are more than 16,000 packages listed on CRAN for basically every task you can imagine.
